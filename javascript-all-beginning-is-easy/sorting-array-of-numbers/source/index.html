Exercise

Rewrite the JavaScript sort so that the following array of numbers will be sorted ascendingly.

[16,10,12,2,1]

Results

Calling native sort

Sort sorts numbers lexicographically therefore 10 follows 1 and 12 follows 10 as 0 is lexicographically ordered before 2

[16,10,12,2,1].sort();

Calling new sort function

Sort awaits a compare function with the following rules:

  • 0: if a and b are equal
  • > 0: if a is bigger than b
  • < 0: if a is smaller than b
function _sortAscending(a,b) {
    return a - b;
}

function _sortNumersAscending(numbers) {
    return numbers.sort(_sortAscending);
}
