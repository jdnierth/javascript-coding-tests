Task

  • Write a function quickSort(https://btholt.github.io/four-semesters-of-cs/)
  • Quicksort should grab a pivot from the end and then separate the list (not including the pivot) into two lists, smaller than the pivot and larger than the pivot.
  • Call quickSort on both of those lists independently.
  • Once those two lists come back sorted, concatenate the "left" (or smaller numbers) list, the pivot, and the "right" (or larger numbers) list and return that. The base case is when quickSort is called on a list with length less-than-or-equal-to 1. In the base case, just return the array given.

Notes

  • Choose a pivot element from the end of the list to sort
  • Separate the list (excluding the pivot) into 2 lists
  • O(n log n) but takes up less memory than mergesort.
  • Though it performs poorly if an already sorted list is passed.
