#Q: What do the functions `call`,`bind` and `apply` do?

* They provide a way to stabilise / lock down the value of `this` when executing functions.

#Q How can you call a function? 
##Using Braces
```
function foo() { ... }

foo();
```

## Using call
```
function foo() {...}

foo.call();
```


## Using apply
```
function foo() {...}

foo.apply();
```
