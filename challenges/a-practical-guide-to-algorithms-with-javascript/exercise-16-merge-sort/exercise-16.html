Tasks

  • Make a function mergeSort that sorts a list of numbers.
  • Steps:
    • 1. If the length of the list is < 2, return
    • 2. Break the list into 2 halves (left, right)
    • 3. LeftSort = mergeSort(leftList)
    • 4. RightSort = mergeSort(rightList)
    • 5. Return merge(leftList, rightList)
  • Tip: Split the array into halves and merge them recursively.
  • Compare the arrays item by item and return the concatenated result.

Notes

  • MergeSort divides the array until it has lists of length 1.
  • The time complexity for mergeSort is O(n log n). log n is because of the cutting in halves. n is from merging the results together.
