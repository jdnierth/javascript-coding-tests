Tasks

  • Make a function binarySearch that finds a number n in a sorted array of numbers.
  • The function binarySearch takes 2 arguments: list as the elements to search through and item, the element to search for.
  • Compare the middle of the list with the item that needs to be searched for ...
    • If the elements are not equal, the half in which the target can not lie is eliminated and the search continues on the remaining half.

Notes

  • Binary search, also known as logarithmic search, find a value in a sorted list of elements.
  • Binary search compares the target value to the middle element of the array.
