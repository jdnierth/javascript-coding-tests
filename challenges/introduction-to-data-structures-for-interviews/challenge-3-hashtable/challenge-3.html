HashTable

Implement the data structure of a HashTable. Implement the following functions:

  • insert: Inserts a new key-value pair (constant time)
  • remove: Deletes a key-value pair (constant time)
  • retrieve: Returns the value associated with a key (constant time)

Notes

  • A Hashtable is not ordered, we don't sort them although you could.
  • Good for fast lookups and having flexible keys.
  • Quick insertion with keys that can be more than numbers.
  • Hashing function: Mathematical calculate location of values. Takes the key (string) and hashes it to a numerical value which is an index to an array (under the hood it will be a memory address). A hashing function will always give the same output for the same input.
  • If you are implementing a HashTable once it becomes 50% full, you want to double it, then you rehash the values. The hasFunction should take the size.
