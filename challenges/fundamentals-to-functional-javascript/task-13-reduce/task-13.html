Task 13 - Usage of the reduce function

Use reduce to implement the add function. 

    _.reduce(collection,cb,acc)
Example 1 
    _.reduce([1,5,6],add,0) // 12
Example 2 
    _reduce({a:1,b:2,c:1},function(result,value,key) {
        (result[value] || (result[value] = [])).push(key);
        return result;
    },{});
Example 2 should print out: {'1': ['a','c'], '2':['b']};
