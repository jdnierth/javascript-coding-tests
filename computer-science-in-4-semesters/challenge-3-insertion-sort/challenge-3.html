Insertion Sort

Task

  • Write a function insertionSort
  • It will take in a array of numbers and return a sorted array numbers
  • The idea here is that the beginning of your list is sorted and everything else is assumed to be an unsorted mess.
  • The outer loop goes over the whole list, the index of which signifies where the "sorted" part of the list is.
  • The inner loop goes over the sorted part of the list and inserts it into the correct position in the array.

Notes

  • Insertion sort is useful for nearly sorted datasets.
  • It's worst case scenario is similar to bubble sort.
  • You loop over your unsorted array and put the values in a sorted array.
  • The complexity of insertionSort is O(n2) as you have an outer and inner loop.
    
            [5,3,6] -> [5]
            [5,3,6] -> [3,5]
