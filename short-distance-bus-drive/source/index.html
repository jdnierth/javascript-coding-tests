Exercise

Given the following bus stops 

        "Nordostbahnhof",
        "Theresienkrankenhaus",
        "Teutoburger Str.",
        "Leipziger Str.",
        "Dresdener Str.",
        "Spitalhof",
        "Hubertusstr.",
        "Tattersall",
        "Martha-Maria-Krkhs."

Results

Get the position of the bus stop 'Hubertusstr.' and remove all bus stops after that bus stop from the array.

    Write a function that returns all bus stops between a given start bus stop and a given end bus stop. E.g. From: 'Nordostbahnhof' To: 'Dresdener Str.'
