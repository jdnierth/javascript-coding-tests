Challenge 4

  • The function forEach takes an array and a callback, and runs the callback on each element of the array.
  • forEach does not return anything.

Rebuild map from challenge 3 as mapWith. This time you're going to use forEach inside of mapWith instead of using a for loop. 

   var alphabet = '';
    var letters = ['a', 'b', 'c', 'd'];
    forEach(letters, function(char) {
    alphabet += char;
    });
    console.log(alphabet);   //prints 'abcd'
